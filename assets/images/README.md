# CanCare Logo

## إضافة شعار CanCare

يجب إضافة صورة الشعار هنا باسم `cancare_logo.png`

### المواصفات المطلوبة:
- **اسم الملف**: `cancare_logo.png`
- **الحجم الموصى به**: 
  - للاستخدام في التطبيق: 512x512 بكسل أو أكبر
  - للاستخدام كأيقونة التطبيق: 1024x1024 بكسل
- **الصيغة**: PNG مع خلفية شفافة (transparent background)
- **الوصف**: شعار CanCare يتكون من حرفين C متداخلين (أحدهما أزرق فاتح والآخر بنفسجي) مع شريط الوعي البنفسجي وموجة نبض القلب البيضاء

### الاستخدام:
- يتم استخدام الصورة في:
  1. صفحة تسجيل الدخول (`login_page.dart`)
  2. Drawer في صفحات الطبيب (`doctor_common_widgets.dart`)
  3. أيقونة التطبيق (Android/iOS)

### ملاحظات:
- إذا لم تكن الصورة موجودة، سيتم استخدام الأيقونة الافتراضية `health_and_safety_rounded` كبديل
- تأكد من تحديث `pubspec.yaml` (تم بالفعل)
